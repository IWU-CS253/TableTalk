Iteration Goal:
Establish a high level, well functioning accessible software available to users through browsers, implemented and tested under CRUD standards and full stack work.
 
Planned Tasks:
Backend Development:
- Set up separate SQL databases to hold posts, profiles, and log in information
- Develop retrieval methods of data to display back to the user when requested
- Create end points and navigation routes for users to navigate with the webpage
Front End Development:
	- Develop and customize individual html, css, and python files optimized with javascript
and bootstrap code for each page the user might interact with. For example, layout.html,
cart.html, main_feed.html, style.css, etc.
- Build UI components like the cart page, profile pages, etc.
Integration Development:
	- Connect UI calls and requests with backend data retrieval
	- Add error calls or displays when users request information incorrectly or that they do
not have access to
	- Develop the app.py file to navigate users to new pages and sectors of the application
Testing:
	- Build user tests for each individual page developed
	- Test edge cases and atypical opportunities along with the expected user interactions
	- Log bugs during testing to report back to group in next team meeting
	- Follow and test CRUD operations
